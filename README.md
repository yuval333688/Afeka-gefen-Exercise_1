# The 5 Most Beautiful Sites I Have Visited (or Want to Visit)

This project was created as part of the Web Development course at Afeka College.

The website includes:
- A main homepage (`index.html`)
- Five separate pages, each representing one site:
  - `web1.html`
  - `web2.html` 
  - `web3.html` 
  - `web4.html`  
  - `web5.html` 

Each page includes:
- A simple layout in HTML only (no advanced technologies)
- A title
- A link back to the homepage, as required

## Technologies Used
Only the technologies allowed in the assignment:
- Basic HTML
- Very minimal structure
- No frameworks, no JavaScript, no external libraries
